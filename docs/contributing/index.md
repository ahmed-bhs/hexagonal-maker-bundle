# Contributing

We welcome contributions to Hexagonal Maker Bundle!

## How to Contribute

- [Overview](overview.md) - Contribution guidelines
- [Development](development.md) - Development setup and workflow

## Ways to Contribute

1. **Report Bugs** - Open an issue on GitHub
2. **Suggest Features** - Share your ideas
3. **Submit Pull Requests** - Fix bugs or add features
4. **Improve Documentation** - Help make docs better
5. **Share Examples** - Contribute real-world examples

## Getting Started

Read the [Contributing Overview](overview.md) to get started with contributing to the project.

Thank you for helping make Hexagonal Maker Bundle better!
